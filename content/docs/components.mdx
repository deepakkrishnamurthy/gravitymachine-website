## Installation

Components are provided by the `@reflexjs/components` package.

```jsx
npm i @reflexjs/components
```

Note: `@reflexjs/components` can also be used in any React project.

## What is a component?

A component is a small reusable piece of content that you can use to build your site.

In Reflex we have made it easy for you to build and theme components.

Creating and styling component is as easy as writing HTML.

<P fontSize="xl" fontWeight="semibold">
  If you know HTML you know how to use this.
</P>

Let's take a look.

### HTML to Components

In HTML, to create a button you use the `button` tag. In Reflex, you use `Button`. In HTML, to create a heading, you use `h1`, in Reflex you use `H1`. See what we did there?

Every HTML tag has an equivalent Reflex component. We have built components for almost all HTML tags.

## Build a button

Start with a simple `HTML` button.

<button>A button</button>

```html
<button>A button</button>
```

Let's turn this into a Reflex/React component.

<Button>A button</Button>

```jsx
<Button>A button</Button>
```

This is a button but with power!

## Style a button

Styling is a easy as adding attributes to the component, just like `HTML`.

<ThemeProvider>
  <Button background="tomato">A button</Button>
</ThemeProvider>

```jsx
<Button background="tomato">A button</Button>
```

Now turn this into a fancy button.

<Button
  background="tomato"
  color="white"
  px="16px"
  py="12px"
  border="0"
  fontSize="18px"
  borderRadius="5px"
  hoverBackground="purple"
>
  Fancy Button
</Button>

```js
<Button
  background="tomato"
  color="white"
  padding="12px 16px"
  border="0"
  fontSize="18px"
  borderRadius="5px"
  hoverBackground="purple"
>
  Fancy Button
</Button>
```
