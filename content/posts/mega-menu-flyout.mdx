Start by adding `@reflexjs/gatsby-theme-nav` to your site. **If you're using the base starter, this is already configured for you.**

We are going to build the following nav:

<H2 fontSize="3xl">Demo</H2>

<NavMenu name="example" borderWidth="1" />

<H2 fontSize="3xl">Create a nav</H2>

Create a nav in `content/navs/example.mdx`:

```mdx title=content/navs/example.mdx
- [Home](/)
- [About](/about)
- [Contact](/contact)
```

To render this nav on your site, use a `NavMenu` component.

```jsx title=content/blocks/header.mdx
<NavMenu name="example" />
```

<H2 fontSize="3xl">Add dropdown</H2>

Add a dropdown by using a nested list.

```mdx title=content/navs/example.mdx
- [Home](/)
- Dropdown // highlight-line
  - [Item One](/item-one) // highlight-line
  - [Item Two](/item-two) // highlight-line
  - [Item Three](/item-three) // highlight-line
- [About](/about)
- [Contact](/contact)
```

<H2 fontSize="3xl">Add mega menu</H2>

To create a mega menu, use a block as a dropdown.

Create a block in `content/blocks/example-nav-block.mdx`

```jsx title=content/blocks/example-nav-block.mdx
<Div>
  <Grid p="4" col="1fr 1fr" w="460px" gap="10">
    <Div>
      <H4 my="0" fontSize="md">
        Section One
      </H4>
      <P mt="2" mb="0" fontSize="sm">
        Autem maiores ab praesentium. Blanditiis numquam eaque odit. Corporis
        totam sit aliquid.
      </P>
    </Div>
    <Div>
      <H4 my="0" fontSize="md">
        Section One
      </H4>
      <P mt="2" mb="0" fontSize="sm">
        Autem maiores ab praesentium. Blanditiis numquam eaque odit. Corporis
        totam sit aliquid.
      </P>
    </Div>
  </Grid>
  <Flexbox bg="muted" px="2" py="2">
    <Button variant="link sm">
      Learn more
      <Icon name="arrow-right" ml="2" size="4" lineHeight="1" />
    </Button>
  </Flexbox>
</Div>
```

Next, add this block to your nav.

```mdx title=content/navs/example.mdx
- [Home](/)
- [About](/about)
- Dropdown
  - [Item One](/item-one)
  - [Item Two](/item-two)
  - [Item Three](/item-three)
- Mega menu // highlight-line
  - <Block src="example-nav-block" /> // highlight-line
- [Contact](/contact)
```

That's it. You can learn more about building navs in the [docs](/docs/navs).
